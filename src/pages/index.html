Developer Documentation

Javascript API

Customize Optimizely's behavior on web pages. You can control how Optimizely buckets users into variations, integrate custom analytics tools, or manually activate experiments.

REST API

Manage your Optimizely experiments programmatically. You can modify projects, experiments, and variations from your own code. This makes it easy to create and start experiments directly from a content-management system.

iOS SDK

Create and run experiments in your iOS app. Use our code-free Visual Editor as well as Live Variables and Code Blocks to optimize your app. Straightforward SDK integration with CocoaPods support.

Android SDK

Create and run experiments in your Android app. Use our code-free Visual Editor as well as Live Variables and Code Blocks to optimize your app. Straightforward SDK integration with Gradle support.
